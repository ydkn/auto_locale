# auto_locale

Automatically set the locale from the browsers HTTP_ACCEPT_LANGUAGE header.

## Installation

Add this line to your application's Gemfile:

    gem 'auto_locale'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install auto_locale
